# 🌟 Nova - Your Personal Voice Assistant

Nova is a fast and versatile voice assistant written in Python that can handle daily tasks seamlessly using voice commands.

---

## 🚀 Highlights
- 🔹 Completely written in Python
- 🔹 Integrates **Gemini Pro** via `genai` API
- 🔹 Uses `SpeechRecognition` for voice input
- 🔹 Supports multiple APIs for diverse tasks:
  - Weather updates
  - Messaging on WhatsApp
  - Music playback via Spotify

---

## 🛠️ Getting Started

```bash
$ git clone https://github.com/ananyab1909/Nova-voice-assistant.git
$ cd Nova-voice-assistant
$ pip install -r requirements.txt
$ python main.py
